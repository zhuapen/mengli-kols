小红书卡审词避坑

⚠️ 常见违禁词 →
❌ 绝对化用语：最好、第一、唯一、100%
❌ 医疗用语：治疗、治愈、药效、处方
❌ 虚假宣传：立即见效、永不反弹、根治
❌ 诱导用语：不买就亏、错过再无、限时秒杀
✅ 安全替代表达 →
✓ 最好 → 个人觉得很不错 / 很推荐
✓ 治疗 → 帮助改善 / 辅助调理
✓ 立即见效 → 坚持使用一段时间后
✓ 100% → 大部分 / 绝大多数
